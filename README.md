# 📝 C++ TO-DO APP

A console-based To-Do Application built using modern C++ concepts and Object-Oriented Programming.

This project helped me learn:
- OOP concepts
- Classes & Objects
- Encapsulation
- Vectors
- File Handling
- Serialization / Deserialization
- Menu-driven applications
- Parsing using stringstream
- Persistent storage using text files

---

# ✨ Features

✅ Add Tasks  
✅ View Tasks  
✅ Mark Tasks as Complete  
✅ Delete Tasks  
✅ Save Tasks to File  
✅ Load Tasks from File  
✅ Persistent Data Storage  
✅ Command/Menu Driven Interface  

---

# 🛠 Technologies Used

- C++
- STL Vector
- File Streams (`ifstream`, `ofstream`)
- StringStream
- Object-Oriented Programming
